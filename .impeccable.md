## Design Context

### Users
Crianças e adolescentes de 8–14 anos aprendendo empreendedorismo de forma gamificada. Pais supervisionam via dashboard separado. Uso principalmente em dispositivos móveis, durante o dia ou à noite. Contexto casual/lúdico.

### Brand Personality
Energético, aventureiro, caloroso. Parece um jogo premium — não um app corporativo de edtech. Inspiração: Duolingo meets mobile game.

### Aesthetic Direction
**Dark mode** com fundo azul-escuro profundo (`#0C1222`) + padrão de estrelas/pontos + orbs de cor ambiente (azul + roxo). Tipografia: Grandstander (display, títulos) + Figtree (corpo, labels). Paleta: azul primário (`#4D7FFF`), laranja accent (`#FF8C42`), verde sucesso (`#2EDF74`). Cabeçalhos de módulo em full-bleed colorido com texto branco (estilo "chapter header" de jogo). Cards escuros com sombra e borda sutil rgba.

### Design Principles
1. **Game-first**: cada elemento deve parecer parte de um jogo de aventura, não de um app educacional genérico
2. **Contraste dramático**: headers coloridos sobre fundo escuro criam hierarquia visual imediata
3. **Motion with purpose**: animações já existentes (pop-in, card-rise, node-active) são mantidas — adicionar apenas quando justificado
4. **Tipografia com personalidade**: Grandstander para títulos/headings, Figtree para body — nunca trocar para fontes neutras
5. **Sem border-stripes**: nunca usar `border-left` colorido como accent em cards — usar superfície completa colorida ou shadow
